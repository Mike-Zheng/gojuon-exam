## 五十音の学習ツール

小遊戲性質的羅馬拼音五十音練習。

>2021/06/14
* 遊戲性新介面
* 鍵盤練習為主輸入羅馬英文
* 隨機出題
* 正確失敗提示
* 正確會自動跳題(auto focus)
* 失敗提示後可以透過backspace/del重新輸入
* 忘記了可以按空白鍵投降
* 加入平假名片假名
* 字元支援增強
